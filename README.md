A fun interactive Valentine’s Day webpage built using HTML, CSS, and JavaScript.
This project creates a playful “Will you be my Valentine?” experience where:
The No button changes text every time it’s clicked.
The Yes button gradually grows bigger.
After multiple No clicks, the Yes button takes over the screen.
When Yes is clicked:
A confirmation screen appears.
A response is sent via email using Formspree.
Fetch with fromspeer link...
The back button is controlled until a final decision is made.
